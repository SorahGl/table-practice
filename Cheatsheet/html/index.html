Study this cheatsheet to learn html tags and what they mean.

Tag Name How it looks what it does
Html tag <html>Your code is typed here.</html> Allows computer to understand text as html code.
Body tag <body>Body is typed in this space</body> The body is where the main content belongs in.
Heading 1 <h1>Type heading here.</h1> This is the main heading.
Line break <line break> This causes a break in the paragraph, after the tag, a new paragraphwill start. (no closing tag is neccesary).
